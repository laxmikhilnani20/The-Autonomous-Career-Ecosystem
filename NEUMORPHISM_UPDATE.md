# Neumorphism UI Update Plan

## Design System
- **Background**: Light gray (#e0e0e0, gray-100)
- **Cards**: Soft shadows with embossed effect
  - Outer shadow: `shadow-[8px_8px_16px_#bebebe,-8px_-8px_16px_#ffffff]`
  - Inner shadow (pressed): `shadow-[inset_4px_4px_8px_#bebebe,inset_-4px_-4px_8px_#ffffff]`
- **Colors**: Pastel blues and purples
- **Typography**: Gray-800 for primary, Gray-600 for secondary
- **Buttons**: Gradient (blue-400 to purple-400) with soft shadows
